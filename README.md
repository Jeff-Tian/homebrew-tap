# homebrew-tap
To install anything in here, first run:
```
brew tap spectralkh/tap
```

# Casks

### to
https://github.com/SpectralKH/to

Install by running:
```
brew cask install to
```

### flutter
Install by running:
```
brew cask install flutter
```

Finally, to see if there are dependencies you need to install, run:
```
flutter doctor
```
